# What I've Learned - JuaraAndroid Edition

JuaraAndroid adalah sebuah event yang diselenggarakan oleh Google untuk developer Indonesia yang berfokus pada pembelajaran Android.

Dalam event tersebut, salah satunya adalah developer diminta untuk menghasilkan suatu aplikasi yang dibuat secara mandiri.

Aplikasi ini dibuat sebagai prasyarat menyelesaikan event JuaraAndroid !

## Cara menjalankan aplikasi ini:
1. Clone github ini
1. Buka di dalam android studio
1. Build aplikasi ini

## Catatan:
- Aplikasi ini tidak menggunakan permission apapun, sehingga aman untuk dipasang (bagi para developer yang sangat teliti dengan permission)
- Aplikasi ini tidak menggunakan ads, sehingga bagi yang takut akan ads juga aman untuk build aplikasi ini.

## Malas untuk cloning?

Silakan buka hasil jadi aplikasi ini di [Google Play Store](https://play.google.com/store/apps/details?id=dev.witheredflowers.whativelearned)
